Выберите режим:
Угадай столицу страны.
В этом режиме будет показана страна и 4 разных столицы. Вам надо будет выбрать ту столицу которая соответствует показаной стране.
Угадай страну по столице.
Этом режиме похож на предыдущий только всё наоборот. Вам надо будет выбрать ту страну которая соответствует показаной столице.
Угадай страну по флагу.
В этом режиме вам будет показан флаг. Вам надо будет выбрать ту страну которая соответствует показаному флагу.
Угадай флаг по названию страны.
Этот режим похож на предыдущий только наоборот. Вам надо будет выбрать тот флаг которая соответствует показаной стране.
